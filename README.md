# NBA Game Predictor

A machine learning project that predicts the outcome of NBA games using historical game data and statistical features.

## Overview
This project builds a predictive model that estimates the probability of one NBA team beating another based on past performance data. The goal of the project is to explore how sports analytics and machine learning techniques can be used to forecast game outcomes.

The pipeline includes data collection, preprocessing, feature engineering, model training, and prediction generation.

## Features
- Loads and processes historical NBA game data
- Performs feature engineering using team statistics
- Trains a machine learning classification model to predict game winners
- Evaluates model performance using accuracy and validation metrics
- Generates win probability predictions for matchups

## Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib / visualization tools

## How It Works
1. **Data Collection**
   - Historical NBA game data is imported and cleaned.

2. **Feature Engineering**
   - Team performance metrics are transformed into model inputs.

3. **Model Training**
   - A classification algorithm is trained to predict game outcomes.

4. **Evaluation**
   - Model accuracy and prediction reliability are measured.

5. **Prediction**
   - The model predicts which team is most likely to win a given matchup.
